# 課題10　 -自分のポートフォリオとしてのWebサイト-

## ①課題内容（どんな作品か）
- フレキシブルレイアウト（ブレイクポイント ＋ リキッドレイアウトのこと）と、グリッドレイアウトの練習として制作したWebサイト
- スニーカーの魅力を紹介する架空のWebサイトを想定した
- 前回まで制作していた『ザ・ゲーム・ライオンキング』は、作り込みすぎて逆に方向転換が難しくなった。自分としても作っていて楽しくなくなってきた。なので今回は言語関係無しに、今の自分が一番つくりたいものをつくった

## ②工夫した点・こだわった点
- CSSのプロパティである「object-fit」と「height: calc(100vh - 80px);」を用いて、どんなデバイス幅でもファーストビュー動画がデバイスの高さいっぱいになるようにした点
- Sassのフォルダ構成とCSS設計にこだわった。何十ページもあるような大規模なWebサイトを作るときでも同じようにコードを書けるように、今のうちからしっかりした書き方を意識している
